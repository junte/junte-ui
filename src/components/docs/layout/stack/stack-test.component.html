A stack organizes a list of elements in one vertical or horizontal line using common layout rules (flex layout) and similar space (gutter) between elements.

The stack is the most popular pattern for aligning elements in views.

Стек организует список элементов в одну вертикальную или горизонтальную линию с использованием общих правил компоновки (flex макет) и аналогичное пространство (желоб) между элементами.

Стек - самый популярный шаблон для выравнивания элементов на видах.
