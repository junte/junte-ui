Installation

Install Junte UI with

Once installed you need to import our main module.

Some components use angular animation, you need to import BrowserAnimationsModule to your application.

Or you can import only modules of target components to decrease bundle size.

We didn't compile styles into components, ðŸ”¥ but we save encapsulation using our own approach.
It means you have full control over styles, you can customize, build, switch themes or customize components in the context!

Import style and SVG icon assets file link to angular.json

Import the style files of the components you need into the global style file style.scss

If you need styles of all the components import the file components.scss

If you need common styles import the file common.scss
